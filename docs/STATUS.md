# 当前进度

更新时间：2026-04-11

## 当前定位

项目当前已经从“Word 转 PPT 小工具”转向“PDF 真题整理工具”。

主链路是：

`PDF -> ExamProject -> Word / PPT / JSON`

GUI 也已经同步转成单一 PDF 向导，不再保留单独的 Word -> PPT 页面。

## 已完成

### 1. 统一工程流

- 引入 `ExamProject` 作为统一中间层
- 打通 `PDF -> 工程 -> Word / PPT / JSON`
- 加入科目筛选与题号范围筛选

### 2. GUI 改造

- 改成四步向导
- 支持六大模块多选
- 支持预览结构树
- 支持资料分析材料原貌预览
- 支持局部编辑材料与题目关系

### 3. 六大模块解析

已覆盖：

- 政治理论
- 常识判断
- 言语理解与表达
- 数量关系
- 判断推理
- 资料分析

### 4. 资料分析增强

- 支持材料图片抽取
- 支持文本表格材料按 PDF 区域截图
- 支持 Word / PPT / GUI 复用同一套材料裁图逻辑
- 支持混合文本 + 图表材料

### 5. 解析器修补

已处理的高频边界：

- 题号单独成行
- `66. 题干` 同行切分
- `D` 选项续行
- `D` 选项后直接跟下一题
- 资料分析无“材料二/三/四”时的自动拆组
- 页面图片缺失补提取
- 双栏页误判
- 客观题篇首说明误并入首题

## 当前已知风险

- 扫描版 PDF / OCR 错字仍可能造成错切
- 跨页材料仍未完全稳定
- 极端复杂表格和多栏版式仍可能读序异常
- 工程 JSON 目前偏导出用途，还不是完整回载编辑格式
